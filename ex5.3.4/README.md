# Alyra : Exercice 5.3.4
Créer un jeu de cartes à collectionner décentralisé

## Installer ipfs
```bash
brew install ipfs 
```

## Initialiser et lancer ipfs
```bash
ipfs init
ipfs daemon
```

## Installer http-server
```bash
npm install --global http-server
```

##Lancer http-server
```bash
http-server
```

Ouvrir votre navigateur sur http://127.0.0.1:8081
